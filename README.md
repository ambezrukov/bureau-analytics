# Аналитика юридического бюро

Интерактивный дашборд на Streamlit для анализа данных рабочего времени из ProjectMate.

## Страницы

| Страница | Содержание |
|----------|-----------|
| 📊 Обзор | KPI-карточки, динамика по месяцам, распределение по типам |
| 👥 Сотрудники | Показатели каждого сотрудника за 3 периода с подсветкой |
| 📈 Динамика | Конструктор произвольных графиков |
| 📁 Проекты | Топ-10 проектов по часам и суммам |
| 💰 ФОТ | В разработке |

## Локальный запуск

```bash
cd dashboard
pip install -r requirements.txt
streamlit run app.py
```

## Данные

Файл `bureau_data.sqlite` содержит выгрузки из ProjectMate за все периоды.

Для обновления данных:
1. Положите новый XLS-файл в `data/`
2. Запустите: `python scripts/import_data.py`
3. Скопируйте обновлённую базу: `cp data/bureau_data.sqlite dashboard/`

## Деплой на Streamlit Cloud

1. Создайте репозиторий на GitHub
2. Запушьте папку `dashboard/`
3. Откройте [share.streamlit.io](https://share.streamlit.io)
4. New app → ваш репозиторий → main → `app.py`
