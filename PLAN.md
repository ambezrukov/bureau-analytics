## 🔐 ЭТАП 3 — АВТОРИЗАЦИЯ

**Варианты (в порядке сложности):**

### Вариант A: streamlit-authenticator (быстро, ~2 часа)
- Логин/пароль в `secrets.toml`
- Подходит для 5–15 пользователей
- Не требует базы пользователей

### Вариант B: Keycloak / Google OAuth (надёжно, ~1 день)
- Полноценный SSO
- Роли: администратор, руководитель группы, просмотр

**Роли для бюро:**
- `admin` — видит всё, может загружать данные
- `group_lead` — видит свою группу + форму ввода показателей
- `viewer` — только просмотр

## 🗂️ АРХИТЕКТУРНЫЕ ДЕТАЛИ ДЛЯ ПАМЯТИ

### Почему win32com.client вместо xlrd
XLS-файлы из ProjectMate >2MB не читаются через xlrd (ложно-повреждённые).
Решение работает только на Windows. На VPS (Linux) использовать xlrd или openpyxl.

### COALESCE для группы сотрудника
```sql
COALESCE(t.department, s.work_group) AS work_group_resolved
```
Источник 1: поле `department` в XLS (когда появится в выгрузках ProjectMate)
Источник 2: `work_group` из справочника staff (сейчас основной источник)

### Будущее поле «Подразделение» в XLS
В `scripts/import_data.py` закомментирована строка:
```python
# 'Подразделение': 'department',
```
Когда поле появится в выгрузках — раскомментировать и перезапустить импорт.

### Курсы валют
Сейчас захардкожены в `pages/1_📊_Обзор.py` и `pages/4_📁_Проекты.py`:
