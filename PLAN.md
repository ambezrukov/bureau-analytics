### Почему win32com.client вместо xlrd
XLS >2MB из ProjectMate не читаются через xlrd на Windows (ложно-повреждённые).
На VPS (Linux) xlrd работает нормально — win32com не нужен.
Скрипт импорта нужно адаптировать для Linux при переносе на VPS.
