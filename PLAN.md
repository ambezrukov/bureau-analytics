### Почему win32com.client вместо xlrd
XLS-файлы из ProjectMate >2MB не читаются через xlrd (ложно-повреждённые).
Решение работает только на Windows. На VPS (Linux) использовать xlrd — файлы читаются нормально.
**Важно:** скрипт импорта нужно адаптировать для VPS (убрать win32com, использовать xlrd напрямую).

### decimal.Decimal в новом формате выгрузки (фев 2026+)
```python
import decimal
if isinstance(value, decimal.Decimal):
    return float(value)
```

### Поле «Подразделение» в XLS
С февраля 2026 поле заполнено в выгрузках. Маппится: `'Подразделение' → 'department'`.
Справочник сотрудников нужен только для окладов (этап ФОТ).

### Совместимость сумм
- Старый формат: «Сумма» → `amount`
- Новый формат: «Сумма, учтено» → `amount_recorded`; `amount` = `amount_recorded`

### Git LFS
Настроен для `*.sqlite`. При клонировании: `git lfs pull` для получения БД.
